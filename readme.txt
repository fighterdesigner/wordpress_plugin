Pictures plugin for your theme, you could choose as many images as you want.

== Description ==
This the plugin that would definitely help you create multiple galleries for your theme, all you need to do is install the plugin then give a name to your gallery then choose your pictures at least one, and click create, you are going to see that your plugin is created, click on the arrow down you are going to see two codes the first one if you are a developer copy it buy clicking copy button and the other if you are using elementor or you want to use it in a post or page, do the same thing.

== Installation ==
you could either do two things install it using wordpres.org, or it\'s link, download it then go to plugins, then click on add new then click on upload and select the file you downloaded then click activate.
